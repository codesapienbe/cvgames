About the Project üîç

CVGames is an open-source project dedicated to creating computer vision games that can be played using just a webcam. Our mission is to make gaming accessible to everyone, regardless of physical abilities or special equipment.

The project was started in 2025 and has grown with contributions from developers around the world.
